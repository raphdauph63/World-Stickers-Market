# World Stickers Market

Boutique française spécialisée dans les cartes Panini de la Coupe du Monde 2026.

Fonctionnalités prévues :

- Vente de cartes à l'unité
- Recherche par joueur et numéro
- Gestion du panier
- Calcul automatique des frais de port
- Administration depuis iPhone
- Gestion des commandes
